Sample codes for App Review

Sample QR, Aztec, DataMatrix, and PDF417 codes for testing the Scan Screen QR Code macOS app. This page is hosted at a stable URL and will remain available for future reviews.

How to test

  1. Launch Scan Screen QR Code. It is a menu-bar app with no Dock icon — look for the QR-viewfinder icon in the menu bar at the top-right of the screen.
  2. On the first scan, macOS prompts for Screen Recording permission (used only for on-device detection). Grant it, then scan again.
  3. To scan one specific code: open its full-screen version (linked in the last column) so only that single code is on screen, then click the menu-bar icon → Scan Screen for QR Code.
  4. Or scan a region: with this table visible, click the menu-bar icon → Scan Selected Area for QR Code and drag a box around any one code below.
  5. The decoded text is copied to the clipboard and shown in a brief floating HUD. Compare it to the “Decoded value” column to confirm.

Note: a whole-screen scan returns the first code it finds, so use the full-screen versions when you want a specific result. Codes are generated by QuickChart.
